# 🏆 赏金广场 / Bounty Plaza

> 外部真实赏金任务认领与分发平台
> 从 Immunefi、Code4rena、HackerOne 等平台聚合真实漏洞赏金任务，分发给社区贡献者，修复后提交赚取赏金。

---

## 📋 流程

```
① 管理员发现真实赏金任务
② 在仓库创建 Issue 描述漏洞 + 标 HONEY 赏金
③ 贡献者回复 /claim 锁定优先权（24h 独占）
④ 24h 后开放给所有人提交
⑤ 截止前通过 PR 提交修复
⑥ 管理员从所有提交中选最优提交到赏金平台
⑦ 到账后最优提交者拿 HONEY，其他人拿参与分
⑧ HONEY 累积到门槛后可兑换现金（100 HONEY = $0.01）
```

## 🪙 HONEY 积分

| 项目 | 数值 |
|------|:----:|
| 兑换率 | **100 HONEY = $0.01** |
| 最低兑换 | **10,000 HONEY**（=$1）起兑 |
| 兑换方式 | 创建 Issue 使用模板申请，管理员审核后打款 |
