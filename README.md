## Bringing Up CosyVoice Model Using TTNN APIs

### Prerequisites
- Install the necessary dependencies:
  ```bash
  pip install ttnn-api
  pip install cosyvoice
  ```

### Running the Script
- Run the script to bring up the CosyVoice model:
  ```bash
  python bring_up_cosyvoice.py
  ```

### Running the Tests
- Run the tests to ensure the model runs correctly:
  ```bash
  python -m unittest test_bring_up_cosyvoice.py
  ```
