# 🏆 赏金广场 / Bounty Plaza

> 外部真实赏金任务聚合与分发平台
>
> 从 Immunefi、Code4rena、HackerOne 等平台聚合真实漏洞赏金任务，
> 分发给社区贡献者，**质量最优 + 速度最快 = 赢家**。

---

## 📋 流程

```
① 管理员发现真实赏金任务 → 创建 Issue
② 贡献者查看 Issue → 编写修复代码 → 提 PR
③ 管理员审查所有 PR → 按代码质量评分
④ 质量最高者获胜；同质量时先提交者获胜
⑤ 管理员将最优代码提交到赏金平台
⑥ 到账后获胜者获得 HONEY 积分
⑦ HONEY 累积到门槛后可兑换现金（100 HONEY = $0.01）
```

## 🪙 HONEY 积分

| 项目 | 数值 |
|------|:----:|
| 兑换率 | **100 HONEY = $0.01** |
| 最低兑换 | **10,000 HONEY**（=$1）起兑 |
| 兑换方式 | 创建 Redeem Request Issue，管理员审核后打款 |
